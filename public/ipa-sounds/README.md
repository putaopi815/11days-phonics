# 音标音频目录

请将音标 m4a 文件放在此目录下。

## 文件名说明

- 建议使用 **ASCII 文件名**，便于各环境加载，例如：`g.m4a`、`dh.m4a`、`r.m4a`、`k.m4a`。
- 若你已有 Unicode 文件名（如 `ɡ.m4a`、`ð.m4a`），无需重命名：播放 `g`/`ɡ`、`ð` 时会通过接口自动尝试对应文件名。

## 无声音的 4 个音标

若 **ð、r、k、g** 点击无声音，请确保存在以下文件之一即可：

| 音标 | 主文件名 | 备用文件名（不重命名也可） |
|------|----------|----------------------------|
| g/ɡ  | `g.m4a`  | `ɡ.m4a`                    |
| ð    | `dh.m4a` | `ð.m4a`                    |
| r    | `r.m4a`  | —                          |
| k    | `k.m4a`  | —                          |

完整文件名列表见项目内 `data/ipa-audio-filenames.md`（如有）。
